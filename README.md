# 本レポジトリについて

EKSをTerraformで構築するためのコード一式を格納しています。EKSの前提となるVPCなども任意で作成可能です。

# バージョンについて

以下のバージョンでの動作を確認しています。

- EKS 
- Terraform
  - Terraform    : v0.13.5
  - AWS provider : v3.12.0
